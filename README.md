# Playwright for Angular with Material

#### Documentation
    https://angular.io/
    https://material.angular.io/
    
- Install:
```sh
$ npm install -g @angular/cli   
  -> @angular/cli@12.0.0
```
- Create App Angular:
```sh
$ ng new pw-app
$ cd .\pw-app\
```
- Add Material to project
``` 
$ ng add @angular/material 
```

#### Run Angular server and Tests
```sh
$ ng serve --open
  -> http://localhost:4200/
```
